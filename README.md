AUTONOMOUS DELIVERY AGENT

Over View Of The Project:- This project explores pathfinding algorithms (BFS, UCS, A*) in a grid. It models a world with static and moving obstacles, allowing an agent to plan and replan its path to a goal. It also includes tools to test and compare how efficiently these algorithms work on different maps.

Features:-

Grid Environment: A navigable 2D map with different terrains and obstacles.
Dynamic Obstacles: Moving blockages that change position.
Pathfinding Algorithms: Implements BFS, UCS, and A* to find paths.
Agent Replanning: The agent dynamically adjusts its path if obstacles appear.
Performance Metrics: Measures algorithm speed, path cost, and efficiency.
Map Generation: Creates various test maps.
Visualization: Plots experiment results for comparison.
